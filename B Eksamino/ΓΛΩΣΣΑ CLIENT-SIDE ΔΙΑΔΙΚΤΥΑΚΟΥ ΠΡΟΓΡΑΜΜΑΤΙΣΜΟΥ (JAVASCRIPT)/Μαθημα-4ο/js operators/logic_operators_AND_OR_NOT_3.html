Οι λογικοί τελεστές AND, OR, NOT

Οι λογικοί τελεστές χρησιμοποιούνται για τον συνδυασμό των λογικών εκφράσεων.Οι λογικοί τελεστές μπορούν να χρησιμοποιηθούν για την τροποποίηση των αποτελεσμάτων των συγκρίσεων.Συνήθως, θα χρησιμοποιήσετε έναν τελεστή σύγκρισης για να ελέγξετε μια συνθήκη και έναν λογικό τελεστή για να συνδυάσετε συνθήκες σε πιο σύνθετη λογική.

Ο τελεστής && επιστρέφει true αν και οι δύο εκφράσεις είναι αληθείς, διαφορετικά επιστρέφει false:

Ο τελεστής || επιστρέφει true αν μία ή και οι δύο εκφράσεις είναι αληθείς, διαφορετικά επιστρέφει false:

Ο τελεστής NOT (!) επιστρέφει true για ψευδείς εκφράσεις και false για αληθείς εκφράσεις.

Ο τελεστής ?? επιστρέφει τον δεξιό τελεστή όταν ο αριστερός τελεστής είναι μηδενικός (null ή undefined), διαφορετικά επιστρέφει τον αριστερό τελεστή.Μια μεταβλητή μπορεί να φιλοξενεί μια τιμή η οποία μπορεί να είναι 0 ή να είναι μια κενή συμβολοσειρά,false,Nan(Not a number).Οταν θέλουμε να ελεγξουμε εαν μια μεταβλητή ειναι μηδενική (null ή undefined) χρησιμοποιούμε τον μηδενικό τελεστή συνένωσης
