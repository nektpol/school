Οι συμβολοσειρές είναι κείμενα που βρίσκονται μεταξύ απλών, διπλών ή αντίστροφων εισαγωγικών(οξείες). Για να δηλώσουμε μια συμβολοσειρά, χρειαζόμαστε ένα όνομα μεταβλητής, έναν τελεστή ανάθεσης και μια τιμή μεταξύ απλών, διπλών ή αντίστροφων εισαγωγικών(οξείες).
