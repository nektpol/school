Eιδη output-confirm() method

Η μέθοδος confirm() εμφανίζει ένα παράθυρο διαλόγου με ένα συγκεκριμένο μήνυμα, μαζί με ένα κουμπί OK και ένα κουμπί Cancel. Ένα παράθυρο επιβεβαίωσης χρησιμοποιείται συχνά για να ζητήσει άδεια από έναν χρήστη για να εκτελέσει κάτι. Η μέθοδος Window confirm() δέχεται μια συμβολοσειρά ως όρισμα. Κάνοντας κλικ στο κουμπί OK επιστρέφεται η τιμή true, ενώ κάνοντας κλικ στο κουμπί Cancel επιστρέφεται η τιμή false.
