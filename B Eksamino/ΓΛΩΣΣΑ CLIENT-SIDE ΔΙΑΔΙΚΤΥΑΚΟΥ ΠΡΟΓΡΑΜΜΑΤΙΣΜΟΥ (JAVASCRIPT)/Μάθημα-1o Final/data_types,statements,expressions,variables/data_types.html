Τυποι δεδομένων

Οι τύποι δεδομένων είναι οι ταξινομήσεις που δίνουμε στα διαφορετικά είδη δεδομένων που χρησιμοποιούμε στον προγραμματισμό.Στη JavaScript, υπάρχουν επτά βασικοί τύποι δεδομένων:

Αριθμός: Οποιοσδήποτε αριθμός, συμπεριλαμβανομένων των αριθμών με δεκαδικούς αριθμούς: 4, 8, 1516, 23,42.Οι αριθμοί γράφονται με ή χωρίς δεκαδικούς και χωρις μονά ή διπλά εισαγωγικά

Συμβολοσειρά: Οποιαδήποτε ομαδοποίηση χαρακτήρων στο πληκτρολόγιό σας (γράμματα, αριθμοί, κενά, σύμβολα κ.λπ.) που περιβάλλεται από μονά εισαγωγικά: ' ... ' ή διπλά εισαγωγικά " ... ", αν και προτιμούμε τα μονά εισαγωγικά. Μερικοί άνθρωποι θέλουν να σκέφτονται τη συμβολοσειρά ως μια φανταχτερή λέξη για κείμενο.Οι συμβολοσειρές/strings είναι κείμενο, γραμμένο σε διπλά ή μονά εισαγωγικά:πχ var x ="Javascript";

Boolean: Αυτός ο τύπος δεδομένων έχει μόνο δύο πιθανές τιμές — είτε true είτε false (χωρίς εισαγωγικά). Είναι χρήσιμο να σκεφτόμαστε τα boolean ως διακόπτες ενεργοποίησης και απενεργοποίησης ή ως απαντήσεις σε μια ερώτηση «ναι» ή «όχι».

Null: Αυτός ο τύπος δεδομένων αντιπροσωπεύει την σκόπιμη απουσία μιας τιμής και αντιπροσωπεύεται από τη λέξη-κλειδί null (χωρίς εισαγωγικά).

Undefined: Αυτός ο τύπος δεδομένων υποδηλώνεται με τη λέξη-κλειδί undefined (χωρίς εισαγωγικά). Αντιπροσωπεύει επίσης την απουσία τιμής αν και έχει διαφορετική χρήση από τη μηδενική. απροσδιόριστο σημαίνει ότι μια δεδομένη τιμή δεν υπάρχει.

Σύμβολο: Ένα νεότερο χαρακτηριστικό της γλώσσας, τα σύμβολα είναι μοναδικά αναγνωριστικά, χρήσιμα σε πιο σύνθετη κωδικοποίηση. Δημιουργούνται με την χρήση του constructor Symbol

Αντικείμενο: Συλλογές σχετικών δεδομένων.

Οι πρώτοι 6 από αυτούς τους τύπους ονομάζονται πρωτόγονοι (primitive) τύποι δεδομένων. Είναι οι πιο βασικοί τύποι δεδομένων στη γλώσσα. Τα αντικείμενα είναι πιο περίπλοκα και θα μάθετε πολύ περισσότερα για αυτά καθώς προχωράτε μέσω της JavaScript. Στην αρχή, οι επτά τύποι μπορεί να μην φαίνονται τόσο πολλοί, αλλά σύντομα θα παρατηρήσετε ότι ο κόσμος ανοίγει με δυνατότητες μόλις αρχίσετε να αξιοποιείτε τον καθένα. Καθώς μαθαίνετε περισσότερα για τα αντικείμενα, θα μπορείτε να δημιουργείτε πολύπλοκες συλλογές δεδομένων.

Η JavaScript χρησιμοποιεί αριθμητικούς τελεστές ( + - * / ) για τον υπολογισμό τιμών και έναν τελεστή εκχώρησης/ανάθεσης ( = ) για να εκχωρήσει τιμές τόσο σε παραστάσεις όσο και σε μεταβλητές: (5 + 6) * 10
