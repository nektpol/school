Συνταξη και ορισμός δηλωσεων στην javascript

δηλώσεις - statements

Ένα πρόγραμμα είναι μια λίστα με "εντολές" που πρέπει να "εκτελεστούν" από έναν υπολογιστή.Aυτές οι εντολές προγραμματισμού ονομάζονται δηλώσεις.Οι δηλώσεις JavaScript αποτελούνται από:Τιμές, τελεστές, εκφράσεις, λέξεις-κλειδιά και σχόλια. Τα ερωτηματικά χωρίζουν τις δηλώσεις JavaScript μεταξύ τους, και προσθέτονται στο τέλος κάθε εκτελέσιμης πρότασης.

Ένα πρόγραμμα JavaScript είναι μια λίστα δηλώσεων προγραμματισμού.Tα προγράμματα JavaScript εκτελούνται από τους περιήγητες Ιστού.

Οι αριθμοί γράφονται με ή χωρίς δεκαδικούς και χωρις μονά ή διπλά εισαγωγικά

Οι συμβολοσειρές/strings είναι κείμενο, γραμμένο σε διπλά ή μονά εισαγωγικά:

Η JavaScript χρησιμοποιεί αριθμητικούς τελεστές ( + - * / ) για τον υπολογισμό τιμών και έναν τελεστή εκχώρησης ( = ) για να εκχωρήσει τιμές σε μεταβλητές::

Javascript εκφράσεις

Μια έκφραση είναι ένας συνδυασμός τιμών, μεταβλητών και τελεστών, η οποία υπολογίζει μια τιμή.

Τα αναγνωριστικά χρησιμοποιούνται για την ονομασία μεταβλητών και συναρτήσεων.Οι κανόνες για τα νομικά ονόματα είναι οι ίδιοι στις περισσότερες γλώσσες προγραμματισμού.Ένα όνομα JavaScript πρέπει να ξεκινά με: 

Ένα γράμμα (A-Z ή a-z)
								Σήμα δολαρίου ($)
								Ή μια κάτω παύλα (_)
