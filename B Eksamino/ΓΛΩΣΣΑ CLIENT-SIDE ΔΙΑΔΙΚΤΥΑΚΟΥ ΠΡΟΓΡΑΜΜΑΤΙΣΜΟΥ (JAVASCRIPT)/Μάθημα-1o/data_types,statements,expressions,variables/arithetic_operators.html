Javascript Operators-Τελεστές Αριθμητικοί

Η Javascript χρησιμοποιεί αριθμητικούς τελεστές για να υπολογίσει τιμές

  • "+" πρόσθεση
  • "-" αφαίρεση
  • "* πολλαπλασιασμός
  • "/" Διαίρεση
  • "**" εκθετικός τελεστής
  • "++" αυξητικός τελεστής κατα 1
  • "-- μειωτικός τελεστής κατα 1

Πρόσθεση

Ο τελεστής (+) προσθέτει αριθμούς.Παραδειγμα 5+6

Αφαίρεση

Ο τελεστής (-) αφαιρεί εναν αριθμό απο έναν άλλο.Παραδειγμα 5-6

Πολλαπλασιασμός

Ο τελεστής (*) πολλαπλασιάζει αριθμούς.Παραδειγμα 5*6

Διαίρεση

Ο τελεστής (/) διαιρεί αριθμούς.Παραδειγμα 5/6

O αυξητικός τελεστής κατα 1

Ο τελεστής (++) αυξάνει αριθμούς κατα 1.Παραδειγμα το 11 γινεται 12

O μειωτικός τελεστής --

Ο τελεστής (--) μειώνει τους αριθμούς κατα 1.Παραδειγμα το 9 γινεται 8

ο εκθετικός τελεστής

για να υπολογίσουμε μια δύναμη, γραφουμε χ2. Για να την υπολογίσουμε γράφουμε στην javascript την έκφραση x**2, όπου x=5

Υπολογισμός του (5+6)*10

Υπολογισμός του "(5+6)*10" με την χρήση μεταβλητών "(x+y)*z"

Javascript Expressions-Εκφράσεις

Μια έκφραση είναι ένας συνδιασμός τιμών, μεταβλητών και τελεστών, τα οποία υπολογίζουν μια τιμή.Ο υπολογισμός(computation) ονομάζεται εκτίμηση(Evaluation)

Παράδειγμα έκφρασης με την χρήση μεταβλητών τιμών:x**2..δύναμη του x στο τετράγωνο
