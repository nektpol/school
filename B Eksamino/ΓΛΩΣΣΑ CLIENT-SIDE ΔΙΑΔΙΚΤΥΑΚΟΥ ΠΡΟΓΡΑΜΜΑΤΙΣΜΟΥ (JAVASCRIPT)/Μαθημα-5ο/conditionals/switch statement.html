H δήλωση switch

Χρησιμοποιήστε τη δήλωση switch για να επιλέξετε ένα από τα πολλά μπλοκ κώδικα που θα εκτελεστούν.

Σύνταξη switch

switch(έκφραση) {
case x:
//τμήμα κώδικα
break;
case y:
//τμήμα κώδικα
break;
default:
//τμήμα κώδικα
}

με αυτό τον τρόπο δουλεύει:
Η έκφραση του switch αξιολογείται μία φορά.
Η τιμή της έκφρασης συγκρίνεται με τις τιμές της κάθε περίπτωσης.
Εάν υπάρχει αντιστοιχία, εκτελείται το συσχετισμένο μπλοκ κώδικα.
Εάν δεν υπάρχει αντιστοίχιση, εκτελείται το προεπιλεγμένο μπλοκ κώδικα.
